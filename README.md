# my vim config

## prerequisites

### git

ubuntu

```
sudo apt-get install git
```

centos

```
sudo yum install git
```

### ctags

ubuntu

```
sudo apt-get install ctags
```

centos

```
sudo yum install ctags
```

## Installation

download this project and run the automatic script `./auto.sh`

copy `my-vim-config/.vimrc` to `~/`

## Shortcut key

F10: paste mode

ctrl+n: open nerdTree

ctrl+t: open taglist
